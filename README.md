# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

1. Assets - add images, videos, documents and more 
2. Common - create common components under this folder.
3. Components - create the components 
4. Pages - create the pages relavent to the components and common components.
5. Hooks - relevant to pages and components/ common components.
6. routes - page urls under private and public routes and import the pages here.
7. utils - create the common functions under this .
8. services - add all api services function pages on components files.
9. config - create url config and asios config under this folder. and env urls also..
10. Entire project use modular css or mui [install the relavent packages.]
